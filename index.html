Setup Guide

Get started in under 5 minutes — no coding required.

1

Create a Metafield definition

In your Shopify Admin go to Settings → Custom data → Collections and click Add definition.

2

Name it exactly "Subcollections"

The name must be Subcollections exactly (case-sensitive).

3

Set type to Collection – List of values

Click Select type, choose Collection, and select List of values (not "One value").

4

Enable Storefront API access

In the Access section, make sure Storefront API access is checked before saving.

5

Add subcollections to your collections

Open a collection, scroll to the Metafields section, and select one or more subcollections.

6

Add the app block in Theme Editor

Go to Online Store → Themes → Customize, navigate to a collection page, and add the Category Tiles block.
